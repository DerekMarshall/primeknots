# viz/

Static HTML + D3 viewers (ARCHITECTURE.md §5). Each viewer is a single file that
renders from emitted JSON only — no computation in JavaScript, no data inlined
into HTML. `viz/data/` holds the emitted JSON and is gitignored (regenerated by
`at emit`).

Viewers are added with the stage that produces their data (`linking.html` with
Stage 1, `borromean.html` with Stage 2, and so on). Stage 0 emits nothing, so
there are no viewers yet.
